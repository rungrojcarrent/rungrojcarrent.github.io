# sparesos
## GitHub Pages 
---

### การเริ่มต้นใช้งาน

1. สมัครบัญชี GitHub หากยังไม่มี และยืนยันอีเมลของคุณ (สำคัญ!)
2. คลิกปุ่ม "Use this template" ที่มุมขวาบน
3. ในหน้า "New repository" ให้ตั้งชื่อ repository เป็น "[ชื่อผู้ใช้ GitHub ของคุณ].github.io" ซึ่งจะเป็น URL ของเว็บไซต์คุณ
4. ตั้งค่าการกำหนดค่าเว็บไซต์และเพิ่มเนื้อหา
5. อัปโหลดไฟล์ใด ๆ (เช่น PDF, .zip) ไปที่โฟลเดอร์ `files/` ไฟล์เหล่านี้จะปรากฏที่ https://[ชื่อผู้ใช้ GitHub ของคุณ].github.io/files/example.pdf
6. ตรวจสอบสถานะได้ที่การตั้งค่า repository ในส่วน "GitHub Pages"
7. (ตัวเลือก) ใช้ Jupyter notebooks หรือสคริปต์ Python ในโฟลเดอร์ `markdown_generator` เพื่อสร้างไฟล์ markdown สำหรับสิ่งพิมพ์และการบรรยายจากไฟล์ TSV

ดูข้อมูลเพิ่มเติมได้ที่ [academicpages.github.io](https://academicpages.github.io/)

---

จากเนื้อหาในไฟล์ `README.md` สามารถเปลี่ยนเป็นภาษาไทยได้ในส่วนที่เป็นคำอธิบายหรือคำแนะนำการใช้งาน **ทม ทองธรรมชาติ**
